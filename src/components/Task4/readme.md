You are building a simple blog app. The app has two pages: 
Page 1 — Home (/) Display a hardcoded list of 3–5 blog posts. 
Each post has an id, title, and excerpt. Render them as cards. 
Each card has a "Read More" button that navigates to the post's detail page.

 Page 2 — Post Detail (/posts/:id) Read the id from the URL, 
 find the matching post from your data, and display its full content (you can make up a body field). 
 If the id doesn't match any post, show a "Post not found" message.
  Also add a "← Back to Home" link on the detail page.
